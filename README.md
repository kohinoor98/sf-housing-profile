# SF Housing Profile

A single-page application showcasing Kohinoor Chatterjee's housing profile for the Mission District, San Francisco.

**Live:** [kohinoor98.github.io/sf-housing-profile](https://kohinoor98.github.io/sf-housing-profile/)

## Stack

- React + TypeScript + Vite
- pnpm
- GitHub Pages (auto-deploy via GitHub Actions)

## Development

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
pnpm preview
```
